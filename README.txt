Purpose
This code is to model single and active site deactivation (originally intended for zeolites, but could be applied to more).

Derivation
The derivation of the equations used in the code can be found in "191124 Hyperbolic deactivation model derivation.pptx"

Use
The setup and tutorial for how to use the script and output can be found in "UsingCatDeactPython.pptx"
